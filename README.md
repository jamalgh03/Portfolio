# Portfolio

## Project Overview
This repository contains a personal portfolio website that presents my professional profile, skills, projects, and contact information.  
The portfolio is designed to showcase work experience, technical skills, and examples of projects in a clean and organized way.

It provides visitors with a quick understanding of who I am, what I do, and how to contact me.

---

## Features
- Introduction and personal summary
- Skills section
- Projects showcase with links and descriptions
- Contact information
- Responsive layout for different screen sizes

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Responsive design principles

---

## How to Use
To view the portfolio:
1. Clone the repository:

git clone https://github.com/jamalgh03/Portfolio.git
2. Open the project folder
3. Open `index.html` in a web browser
